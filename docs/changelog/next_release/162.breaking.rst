Change response schema of ``GET /jobs`` from:

.. code:: python

    {
        "meta": {...},
        "items": [
            {
                "kind": "JOB",
                "id": 123,
                "name": "abc",
                # ...
            }
        ],
    }

to:

.. code:: python

    {
        "meta": {...},
        "items": [
            {
                "id": "123",
                "data": {
                    "id": "123",
                    "name": "abc",
                    # ...
                },
            }
        ],
    }


This makes API response more consistent with others (e.g. ``GET /runs``, ``GET /operations``).
